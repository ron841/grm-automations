### How to update this inventory

After Manus renders the final approved version:
1. Open the PDF and screenshot or describe each photo on every page.
2. Replace "TBD" descriptions with the actual photo content.
3. Add the Pexels or Unsplash photo ID if available.
4. Mark the reuse status as LOCKED.

Before building any new PDF (LISTINGS, FOLLOWUP, NEWOWNER, PLAYBOOK):
1. Read this inventory first.
2. Reject any photo that matches a description already locked to another document.
3. After the new PDF is approved, add its photo inventory below.
