# GRM Design Review — Pre-Build Checklist

Run this before every Manus or Canva session.

### Colors

- [ ] #090c0b, #00b7ce, #FAF8F4, #FFFFFF only — no gold, no green

### Logo

- [ ] Logo_Get_Rooted_Media-01.svg, 220px wide, 24px from left, 24px from top, all paths white
- [ ] Zoom in — confirm tree symbol AND wordmark both visible

### Photography

- [ ] Hero image is local/Ocala-feel — not generic American stock
- [ ] This PDF uses a different hero than every other GRM PDF
- [ ] Cover title is inside the dark bar — not overlaying the photo
- [ ] Media kit pages use local lifestyle photo — not the media kit graphic

### Layout

- [ ] No half-page is more than 30% empty without intentional content
- [ ] Empty space filled with: photo, pull quote, stat block, or design element
- [ ] Callout boxes anchor to BOTTOM — never top
- [ ] Editorial crops are full bleed or cleanly framed — never half-cropped
- [ ] Back cover: logo + tagline + contact — no empty black space
- [ ] Three-brand hierarchy: GET ROOTED MEDIA on top, publication title below
